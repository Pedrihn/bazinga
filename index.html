Create

No SQL create é o comando que cria básicamente tudo desde do começo do projeto, cria o banco de dados e cria a tabela onde os valores serão inseridos.

Read

Read ou 'Ler' na tradução literal, nada mais é que o comando 'SELECT' no SQL, ele te permite ver todos os dados presentes em uma tabela e filtrá-los como queira.

Update

O Update permite a edição ou modificação de itens já presentes em tabelas, é com esse comando que você é permitido a mudar seu nome de usuário em uma rede social.

Delete

Esse comando te deixa excluir valores e até tabelas que não queira mais utilizar, ele pode ser usado para apagar um usuário que está inativo à muito tempo.
